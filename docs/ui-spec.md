## 3. 各セクション詳細

### 3.1 ヘッダー
- サービスロゴ（テキストベース、アイコン + "Slide2Video"）
- シンプルに1行。ナビゲーションなし（1画面なので不要）

### 3.2 キャッチコピー
- メインコピー: 「PDFスライドから動画を自動生成」
- サブコピー: 「AIが台本を作成し、自然な音声でプレゼン動画を生成します」

### 3.3 ① PDFアップロードエリア
- ドラッグ&ドロップ対応の点線枠
- クリックでファイル選択ダイアログも可
- 受付形式: .pdf のみ
- 最大サイズ: 50MB
- ファイル選択後: ファイル名とサイズを表示、×ボタンで取り消し可能

### 3.4 ② 設定エリア
- PDFが選択された後に表示（またはアコーディオンで初期非表示）

| フィールド | 種類 | 選択肢 | デフォルト |
|-----------|------|--------|-----------|
| AIプロバイダ | セレクト | Gemini / OpenAI | Gemini |
| APIキー | パスワード入力 | — | 空欄（必須） |
| 音声エンジン | セレクト | Edge-TTS（無料） / OpenAI TTS | Edge-TTS |
| 話者 | セレクト | Nanami(女性) / Keita(男性) / Aoi(女性) / Daichi(男性) | Nanami |

- 音声エンジンで「OpenAI TTS」を選択した場合、話者の選択肢が OpenAI 用（alloy, echo, nova 等）に切り替わる
- APIキーフィールドには目のアイコンで表示/非表示切り替え
- 「APIキーはサーバーに保存されません」の注記を表示

### 3.5 生成ボタン
- テキスト: 「🎬 動画を生成する」
- PDF未選択 or APIキー未入力時は disabled
- 処理中は「生成中...」+ スピナーに変化し、再押下不可

### 3.6 ③ 進捗エリア
- 生成ボタン押下後に表示される
- プログレスバー: 0〜100%、色はステップに応じて変化
- ステータステキスト: 「PDF解析中...」「スライド 2/5 の台本生成中...」等
- ログエリア: 折りたたみ可能、各ステップのタイムスタンプ付きログ
- エラー時: 赤枠でエラーメッセージを表示、「再試行」ボタン

### 3.7 ④ ダウンロードエリア
- 動画生成完了後に表示される
- 「📥 動画をダウンロード」ボタン
- 「有効期限: 30分」の注記
- 「新しい動画を生成する」リンク（フォームをリセット）

## 4. 状態遷移

    [初期状態]
        │ PDF選択
        ▼
    [ファイル選択済み]
        │ APIキー入力 + 生成ボタン押下
        ▼
    [処理中]
        │  ├── 成功 → [完了]
        │  └── 失敗 → [エラー]
        ▼
    [完了]
        │ 「新しい動画を生成する」
        ▼
    [初期状態に戻る]

    [エラー]
        │ 「再試行」
        ▼
    [処理中]

## 5. レスポンシブ対応

- モバイル（〜768px）: 1カラム、各セクションが縦に積まれる
- デスクトップ（768px〜）: 中央寄せの最大幅640px程度のカード型レイアウト
- 1画面完結のため複雑なレスポンシブ対応は不要
